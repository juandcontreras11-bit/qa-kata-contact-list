# QA Kata Contact List

## Objetivo

Automatizar pruebas UI y API para la aplicación Contact List.

## Tecnologías

- Java
- Serenity BDD
- Rest Assured
- Gherkin
- Screenplay Pattern

## Casos priorizados

### UI
- Login exitoso
- Crear contacto
- Eliminar contacto

### API
- Crear usuario
- Login
- Crear contacto

## Estrategia

Se utilizó una estrategia basada en riesgo, priorizando los procesos críticos de autenticación y gestión de contactos.
